# Stanford Code in Place

Welcome to my Stanford Code in Place repository.

I was selected to participate in Stanford University's Code in Place program from a global pool of more than 66,000 applicants. This repository documents my work strengthening my fluency in Python, computational thinking, and problem-solving through Stanford’s CS106A-based curriculum.

## About This Repository

This repo contains:
- Python exercises and programming challenges
- Algorithms and computational thinking practice
- Debugging experiments and solutions
- Notes and technical exploration
- Small projects and coding exercises

## Why This Matters to Me

As someone working across data science, bioinformatics, AI, and computational biology, I wanted to build deeper fluency in Python beyond relying on Stack Overflow, documentation, and AI-assisted workflows alone.

This repository reflects my effort to strengthen core programming intuition, write cleaner code, and better understand the underlying logic behind the tools and systems I use.

## About Code in Place

Code in Place is Stanford University's global introductory programming program based on Stanford’s CS106A course. The program brings together participants from around the world to strengthen foundational computer science skills through collaborative learning and hands-on programming.
